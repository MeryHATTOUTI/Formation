Les étapes de réception

Décharger le camion : Décharger les conteneurs de déchets du camion à leur arrivée.

Placer sur chariot : Transférer les conteneurs déchargés sur des chariots de transport.

Peser sur la balance : Prendre le poids de chaque lot de déchets sur la balance.

Transvaser dans les grandes bennes : Verser les déchets collectés dans les grandes bennes pour traitement.

Transporter les grandes bennes vers la chambre froide : Déplacer les grandes bennes vers la chambre froide.

Transporter les conteneurs vides vers la phase de désinfection

Acheminer les conteneurs vides vers la zone de lavage.

Séchage et contrôle visuel : Vérifier l'état des conteneurs lavés.

Placer dans la zone de préparation : Disposer les conteneurs désinfectés dans la zone de stockage.
